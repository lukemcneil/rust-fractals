# rust-fractals

Controls
* `w or up` - move up
* `a or left` - move left
* `s or down` - move down
* `d or right` - move right
* `i` - zoom in
* `o` - zoom out
* `]` - draw more pixels
* `[` - draw fewer pixels
* `=` - run with more iterations, draw with more precision
* `-` - run with less iterations, draw with less precision
* `period` - increase coloring constant
* `comma` - decrease coloring constant
* `1` - mandlebrot
* `2` - julia
* `3` - burning ship
