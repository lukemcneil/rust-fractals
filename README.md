# rust-fractals

Controls
* `w` - move up
* `a` - move left
* `s` - move down
* `d` - move right
* `i` - zoom in
* `o` - zoom out
* `.` - make the window bigger
* `,` - make the window smaller
* `=` - run with more iterations, draw with more precision
* `-` - run with less iterations, draw with less precision
